##原神启动器#############\n
这是一个基于C#的Windows Forms应用程序，用于启动原神游戏。

安装和运行----------------------
首先，确保您已经安装了C# 8.0或更高版本。
运行应用程序，即可启动原神游戏。
功能---------------------
该应用程序包含以下功能：

启动游戏
保存游戏和配置文件目录
切换服务器

主要类
MainWindow：应用程序的主要窗口，用于显示游戏和提供用户交互。
Program：应用程序的主类，用于处理游戏启动、保存、TKD服务器切换和B服切换等操作。
主要方法
Main：启动应用程序的事件方法，用于创建窗口和启动游戏。
